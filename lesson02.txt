2. Выяснить, для чего предназначена команда cat. Используя данную команду, создайте два файла с данными, 
а затем объедините их в один. Просмотрите содержимое созданного файла. Переименуйте файл, 
дав ему новое имя.
		man cat
		cat > file1.txt
	- Далее вводим: Hello
		cat > file2.txt
	- вводим: world
	- объеденяем файлы в один:
		cat file1.txt file2.txt > file3.txt
		cat file3.txt
		mv file3.txt result.txt

4. В ОС Linux скрытыми файлами считаются те, имена которых начинаются с символа “.”. 
Сколько скрытых файлов в вашем домашнем каталоге? (Использовать конвейер. 
Подсказка: для подсчета количества строк можно использовать wc).
	sudo find . -type f -name ".*" | wc -l
		Получаем количество равным 15.
